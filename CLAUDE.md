# koogAgent_H 프로젝트

## 프로젝트 개요

Koog 프레임워크 기반의 Java 코딩 에이전트

## 기술 스택

- Java 25, Spring Boot (테스트 컨텍스트만)
- Koog AI Agent Framework (Kotlin 라이브러리를 Java에서 사용)
- JUnit 5, AssertJ
- Maven

## 개발 방법론: TDD (테스트 → 구현 → 리팩토링)

모든 기능은 다음 순서로 개발한다.

1. **테스트 먼저**: 기능의 인터페이스와 기대 동작을 테스트로 정의
2. **구현**: 테스트를 통과시키는 최소한의 코드 작성
3. **리팩토링**: 중복 제거, 명명 개선 등 품질 개선 (테스트는 계속 통과해야 함)
